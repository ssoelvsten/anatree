## Usage

To set up [Troupe](github.com/TroupeLang/Troupe), follow its
[*README*](https://github.com/TroupeLang/Troupe/blob/master/README.md).

| Make Target | Description                               |
|-------------|-------------------------------------------|
| `build`     | Build all targets                         |
| `clean`     | Clear out temporary build files and state |
| `test`      | Runs *HUnit* tests                        |
