Detailed Description

template<typename T = std::string, typename Compare = std::less<typename T::value_type>, typename Set = std::unordered_set<T>, typename Map = std::unordered_map<T, T>>
requires std::copyable<T> && std::sortable<typename T::iterator, Compare> && std::equality_comparable<typename T::value_type> && std::totally_ordered<typename T::value_type>
class anatree< T, Compare, Set, Map >

A data structure capable of storing a set of 'std::string' (or similar) data structures, enabling quick access to all 'anagrams' of each word (within or not).

Template Parameters
TType for words, i.e. lists of elemnts. This type should provide the interface of std::string, such as std::basic_string<_> and std::vector<_>.
CompareOrdering of the symbols within each word. The order has a major impact on the size.
SetType to be used for storing and returning sets of words.
MapType to be used for storing pairs T.
